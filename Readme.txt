0. установить https://nodejs.org/en/ если необходимо
1. скопировать репозиторий (client) https://github.com/dmitrypar/contactsbook
2. запустить npm i и дождаться окончания установки
3. запустить командой npm start
4. скопировать репозиторий (server) https://github.com/dmitrypar/json-server-JWT
5. запустить npm i и дождаться окончания установки
6. запустить командой npm run start-auth
7. для входа зарегистрируйтесь или залогиньтесь например под след. данными
